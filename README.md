# 三效蒸发加热室健康度评估系统 - 设计说明

## 🎯 设计思路

### 核心理念
基于**传热学原理**和**质量守恒定律**，通过实时监测蒸发系统的运行参数，科学评估各效加热室的换热效率，为操作人员提供直观的健康度诊断和投料建议。

### 两大核心功能

#### 1️⃣ 开机投料推荐系统
- **理论依据**：Q = U × A × ΔT（传热公式）+ 质量平衡原理
- **计算逻辑**：基于设备标称热负荷和温差，计算系统峰值脱水能力
- **智能推荐**：根据目标浓度和初始浓度，提供0.75-0.95倍理论能力的安全投料范围

#### 2️⃣ 实时健康度评估
- **自动识别**：通过温度+密度双向插值，自动识别各效出料浓度
- **动态计算**：基于实际流量和浓度变化，计算各效实际蒸发能力
- **健康诊断**：Q_run / Q_set 比值反映换热器结垢程度

## 📊 参数表格详解

### 第一部分：开机投料推荐

| 参数 | 物理意义 | 计算公式 | 工程价值 |
|------|----------|----------|----------|
| **ΣQ_set** | 系统峰值脱水能力 | Σ(Qnom_i × Δt_set_i/Δt_design_i) | 反映系统最大蒸发潜力 |
| **手动输入进料浓度** | 原料液初始浓度 | 操作员手动输入 | 浓缩计算的起点 |
| **理论最大投料量** | 最大进料流量 | ΣQ_set × 目标浓度/(目标-初始) | 系统理论极限 |
| **推荐投料范围** | 安全高效区间 | 0.75～0.95倍理论最大值 | 平衡效率与安全 |
| **建议设定值** | 最优经济点 | 0.90倍理论最大值 | 经济性最佳 |

## ⚙️ 物理过程

```
进料浓度 → [加热室] → 出料浓度
    ↓              ↓
  Q理论 ←→ Q实际 ←→ 健康度
```

1. **热量传递**：蒸汽→加热室壁→料液
2. **质量平衡**：进料=出料+蒸发量
3. **浓度变化**：溶质守恒，水分减少→浓度升高
4. **效率评估**：实际性能/理论性能
